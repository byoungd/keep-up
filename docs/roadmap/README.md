# Roadmap Index

This folder contains product, architecture, and capability roadmaps across runtime, UI, and
cowork initiatives. Use this index to navigate the active tracks and supporting specs.

## Status Vocabulary
- Proposed: Drafted, pending confirmation or resource allocation.
- Ready: Scoped and approved to start once dependencies are cleared.
- Approved: Green-lit for execution (implementation may be ongoing).
- Active: In progress with ongoing updates.
- Completed: Delivered and verified.
