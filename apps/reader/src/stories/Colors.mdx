# Color Palette

Our design system uses CSS custom properties for theming.

## Core Semantic Colors

| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--color-background` | `#ffffff` | `#09090b` | Page background |
| `--color-foreground` | `#0f172a` | `#fafafa` | Primary text |
| `--color-primary` | `#0f172a` | `#fafafa` | Primary actions |
| `--color-secondary` | `#f1f5f9` | `#27272a` | Subtle UI surfaces |
| `--color-muted` | `#f1f5f9` | `#27272a` | Subtle backgrounds |
| `--color-muted-foreground` | `#64748b` | `#a1a1aa` | Secondary text |
| `--color-card` | `#ffffff` | `#09090b` | Card backgrounds |
| `--color-popover` | `#ffffff` | `#09090b` | Popover surfaces |
| `--color-border` | `#e2e8f0` | `#27272a` | Dividers and borders |
| `--color-ring` | `rgba(15, 23, 42, 0.1)` | `rgba(250, 250, 250, 0.2)` | Focus rings |

## Status Colors

| Token | Usage |
|-------|-------|
| `--color-success` | Success states and confirmations |
| `--color-warning` | Warning states and caution banners |
| `--color-info` | Informational states |
| `--color-error` | Error states |
| `--color-destructive` | Destructive actions |

## Surface Colors

| Token | Purpose |
|-------|---------|
| `--color-surface-0` | Base layer |
| `--color-surface-1` | Elevated layer |
| `--color-surface-2` | Secondary elevation |
| `--color-surface-3` | Tertiary elevation |

## Canvas + Ambient

| Token | Usage |
|-------|-------|
| `--color-canvas-warm` | Warm reading background |
| `--color-canvas-mint` | Mint reading background |
| `--color-canvas-sepia` | Sepia reading background |
| `--color-canvas-dark` | Dark reading background |
| `--color-ambient-paper` | Ambient radial glow |
| `--color-ambient-emerald` | Ambient radial glow |
| `--color-ambient-blue` | Ambient radial glow |

## Presence Colors

Presence colors are provided as `--color-presence-1` through `--color-presence-9`.

## Usage in CSS

```css
.element {
  background: var(--color-surface-1);
  color: var(--color-foreground);
  border: 1px solid var(--color-border);
}
```

## Usage in Tailwind

```tsx
<div className="bg-surface-1 text-foreground border-border" />
```
