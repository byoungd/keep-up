# Typography

Our typography system uses CSS custom properties and utility classes.

## Font Families

| Token | Font | Usage |
|-------|------|-------|
| `--font-sans` | Inter, Manrope | UI text |
| `--font-serif` | Fraunces | Reading content |
| `--font-mono` | JetBrains Mono | Code |

## Type Scale

| Class | Size | Weight | Usage |
|-------|------|--------|-------|
| `text-xs` | 12px | 400 | Captions |
| `text-sm` | 14px | 400 | Secondary text |
| `text-base` | 16px | 400 | Body text |
| `text-lg` | 18px | 500 | Emphasized text |
| `text-xl` | 20px | 600 | Small headings |
| `text-2xl` | 24px | 700 | Headings |

## Heading Hierarchy

```tsx
<h1 className="text-3xl font-bold tracking-tight">Page Title</h1>
<h2 className="text-2xl font-semibold">Section Title</h2>
<h3 className="text-xl font-medium">Subsection</h3>
```

## Color Usage

```tsx
<p className="text-foreground">Primary text</p>
<p className="text-muted-foreground">Secondary text</p>
<p className="text-destructive">Error text</p>
```
